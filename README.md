## 🧠 How the analysis works

The app combines several parts:

- A vision model finds players and the ball
- ByteTrack helps keep track of movement over time
- Expected goals logic scores shot quality
- Bayesian calibration adjusts player results
- A graph model studies tactical links between players and zones
- Streamlit displays the results in a simple interface

You do not need to manage each part yourself. The app brings them together for you.
